tape 2: les classifieurs

Un classifieur (binaire dans ce cours) permet, pour chaque point d'entre donn, de calculer un score (qui est une valeur de $\mathbb{R}$) pour dterminer la classe d'un exemple. Cette classe est donne par le signe de ce score: si le score est strictement ngatif, la donne est associe la classe (ou label) $-1$, sinon elle est associe classe (ou label) $+1$.

Reprsentation oriente objets

En LU3IN026, pour implmenter nos algorithmes d'apprentissage, nous allons utiliser quelques lments de programmation oriente objet (POO) qu'autorise le langage Python.

Si vous ne connaissez rien la POO, il est fortement conseill de vous familiariser avec elle, par exemple en commenant par lire une documentation ou suivre un tutorial sur le web (par exemple, https://courspython.com/classes-et-objets.html).

La classe Classifier

Les diffrents classifieurs que l'on va implmenter seront drivs d'une classe unique : la classe Classifier qui est donne ci-dessous. Pour nous, cette classe est l'quivalent une classe abstraite de Java, elle sera drive par hritage pour dfinir des classifieurs particuliers qui seront vus ce semestre (knn, perceptron, arbres de dcision, etc.) et qui devront implmenter toutes les mthodes de cette classe. On aura ainsi un moyen gnrique d'accder nos classifieurs.

La classe Classifier contient 5 mthodes:

  • __init__: (le constructeur) qui permet d'initialiser les paramtres du classifieur. Les classes filles tendront donc cette mthode en rajoutant les paramtres qui les concernent. Le paramtre obligatoire donner la cration d'un classifieur est le nombre de dimensions des donnes qu'il aura traiter et qui peut tre dfini dans cette classe (attribut: self.dimension).
  • train: c'est la mthode qui permet d'entraner le modle, on doit donc donner en argument le dataset qui sert l'entranement.
  • score: cette mthode est utilise une fois le classifieur entrain. Elle prend en argument la description d'un exemple et elle rend un score (qui est donc une valeur de $\mathbb{R}$).
  • predict: cette mthode, comme la prcdente, s'utilise une fois le classifieur entran. Elle rend $\hat y$, le label (-1 ou +1) qui est prdit par le classifieur pour la description de l'exemple donne en argument. Gnralement, cette valeur de prdiction $\hat y$ s'obtient partir du score: si le score est strictement ngatif $\hat y$ vaut $-1$, sinon il vaut $+1$.
  • accuracy: comme les 2 prcdentes, cette mthode s'utilise une fois le classifieur entran. Elle permet de calculer la qualit du classifieur qui est mesure par le taux de bonne classification du classifieur sur le dataset donn en argument. C'est une valeur de $[0,1]$ qui s'obtient divisant le nombre d'exemples du dataset qui sont bien classs par le classifieur par le nombre total d'exemples du dataset.

Les mthodes train, score et predict dpendent du classifieur que l'on implmente, elle ne seront pas dfinies dans la classe Classifier mais dans les classes qui hriteront ce cette classe.

Par contre, la fonction accuracy peut tre dfinie ds maintenant dans cette classe Classifier.

Voici une fonction qui permet de dessiner la frontire de dcision d'un classifieur. La valeur step permet de choisir la prcision de trac du dessin. Le code est entirement fourni mais pour information, l'ide est la suivante:

  1. gnrer une grille de points sur tout l'espace;
  2. valuer le classifieur sur toute la grille;
  3. interpoler un niveau de couleur sur tout l'espace reprsentant les valeurs du classifieur.

def plot_frontiere(desc_set, label_set, classifier, step=30):
    """ desc_set * label_set * Classifier * int -> NoneType
        Remarque: le 4e argument est optionnel et donne la "rsolution" du trac: plus il est important
        et plus le trac de la frontire sera prcis.        
        Cette fonction affiche la frontire de dcision associe au classifieur
    """
    mmax=desc_set.max(0)
    mmin=desc_set.min(0)
    x1grid,x2grid=np.meshgrid(np.linspace(mmin[0],mmax[0],step),np.linspace(mmin[1],mmax[1],step))
    grid=np.hstack((x1grid.reshape(x1grid.size,1),x2grid.reshape(x2grid.size,1)))
    
    # calcul de la prediction pour chaque point de la grille
    res=np.array([classifier.predict(grid[i,:]) for i in range(len(grid)) ])
    res=res.reshape(x1grid.shape)
    # tracer des frontieres
    #colors[0] est la couleur des -1 et colors[1] est la couleur des +1
    plt.contourf(x1grid,x2grid,res,colors=["darksalmon","skyblue"],levels=[-1000,0,1000])

Un classifieur linaire

Nous allons maintenant dfinir un classifieur linaire alatoire en tendant la classe Classifier et en implmentant le constructeur ainsi que les 3 mthodes predict, score et train.

Ce classifieur est un des plus simple que l'on puisse construire: on gnre alatoirement une droite (ou hyperplan si on a plus de 2 dimensions) dans l'espace des dimensions en tirant alatoirement un vecteur $w$ donnant les coefficients de chaque dimensions. L'quation de la droite (hyperplan) est alors donne par $\langle x, w\rangle = 0$ (cf. TD de la semaine 1).

Cela nous permis de construire un vecteur de poids. Cette droite est utilise pour classer les exemples: d'un ct de la droite ils sont prdits de la classe +1, de l'autre ct ils sont prdits de la classe -1.

Naturellement, ce classifieur a de bonnes chances de ne pas tre trs efficace...

Pour un classifieur linaire alatoire:

  • dans le constructeur, on doit gnrer alatoirement un vecteur $w$ de taille input_dimension de valeurs dans $[-1,1]$. Les poids doivent ensuite normaliss (c'est--dire que la norme de $w$ doit tre gale 1). Ainsi, si $v$ est le vecteur alatoire valeurs dans $[-1,1]$ gnr, on obtient un vecteur $w$ normalis partir de $v$ en prenant chaque composante de $v$ est en la divisant par la norme de $v$.
  • pour ce classifieur, la mthode train ne fait rien. On ne mettra qu'un print permettant d'afficher le message "Pas d'apprentissage pour ce classifieur"
  • pour une description d'exemple $x$, la mthode score rend la valeur de $\langle x, w\rangle$
  • pour une description d'exemple $x$, la mthode predict doit utiliser la valeur de $\langle x, w\rangle$ pour prdire le label (+1 ou -1) associ
