Discrtisation des attributs numriques

Comme cela a t dit en cours, discrtiser un attribut numrique consiste en trouver un seuil de coupure dans son ensemble de valeurs aprs les avoir ordonnes.

Ci-dessous, la mthode discretise qui permet de trouver le seuil de coupure optimal dans un ensemble de valeurs associes des classes (labels).

In[10]:
def discretise(m_desc, m_class, num_col):
    """ input:
            - m_desc : (np.array) matrice des descriptions toutes numriques
            - m_class : (np.array) matrice des classes (correspondant  m_desc)
            - num_col : (int) numro de colonne de m_desc  considrer
            - nb_classes : (int) nombre initial de labels dans le dataset (dfaut: 2)
        output: tuple : ((seuil_trouve, entropie), (liste_coupures,liste_entropies))
            -> seuil_trouve (float): meilleur seuil trouv
            -> entropie (float): entropie du seuil trouv (celle qui minimise)
            -> liste_coupures (List[float]): la liste des valeurs seuils qui ont t regardes
            -> liste_entropies (List[float]): la liste des entropies correspondantes aux seuils regards
            (les 2 listes correspondent et sont donc de mme taille)
            REMARQUE: dans le cas o il y a moins de 2 valeurs d'attribut dans m_desc, aucune discrtisation
            n'est possible, on rend donc ((None , +Inf), ([],[])) dans ce cas            
    """
    # Liste trie des valeurs diffrentes prsentes dans m_desc:
    l_valeurs = np.unique(m_desc[:,num_col])
    
    # Si on a moins de 2 valeurs, pas la peine de discrtiser:
    if (len(l_valeurs) < 2):
        return ((None, float('Inf')), ([],[]))
    
    #Initialisation
    best_seuil = None
    best_entropie = float('Inf')
    
    #pour voir ce qui se passe, on va sauver les entropies trouves et les points de coupures:
    liste_entropies = []
    liste_coupures = []
    
    nb_exemples = len(m_class)
    
    for v in l_valeurs:
        cl_inf = m_class[m_desc[:,num_col]<=v]
        cl_sup = m_class[m_desc[:,num_col]>v]
        nb_inf = len(cl_inf)
        nb_sup = len(cl_sup)
        
        # calcul de l'entropie de la coupure
        val_entropie_inf = classif.entropie(cl_inf) # entropie de l'ensemble des inf
        val_entropie_sup = classif.entropie(cl_sup) # entropie de l'ensemble des sup
        
        val_entropie = (nb_inf / float(nb_exemples)) * val_entropie_inf \
                       + (nb_sup / float(nb_exemples)) * val_entropie_sup
        
        # Ajout de la valeur trouve pour retourner l'ensemble des entropies trouves:
        liste_coupures.append(v)
        liste_entropies.append(val_entropie)
        
        # si cette coupure minimise l'entropie, on mmorise ce seuil et son entropie:
        if (best_entropie > val_entropie):
            best_entropie = val_entropie
            best_seuil = v
    
    return (best_seuil, best_entropie), (liste_coupures,liste_entropies)

Une fois un seuil de coupure $s$ trouv sur un attribut $n$, on partitionne la base d'apprentissage en 2 ensembles: - l'ensemble des exemples qui ont une valeur infrieure ou gale $s$ pour l'attribut $n$ - l'ensemble des exemples qui ont une valeur strictement suprieure $s$ pour l'attribut $n$

La fonction partionne ci-dessous prend en argument deux numpy array $m_{desc}$ et $m_{class}$ qui reprsentent une base d'apprentissage, un numro de colonne $n$ et une valeur de seuil $s$ et rend un tuple compos de 2 tuples de numpy arrays:

  • le premier tuple est compos de l'ensemble des descriptions de tous les exemples de $T$ qui ont une valeur pour la colonne $n$ infrieure ou gale $s$ et l'ensemble de leur classe;
  • le second tuple est compos de l'ensemble des descriptions de tous les exemples de $T$ qui ont une valeur pour la colonne $n$ strictement suprieure $s$ et l'ensemble de leur classe;

def partitionne(m_desc,m_class,n,s):
    """ input:
            - m_desc : (np.array) matrice des descriptions toutes numriques
            - m_class : (np.array) matrice des classes (correspondant  m_desc)
            - n : (int) numro de colonne de m_desc
            - s : (float) seuil pour le critre d'arrt
        Hypothse: m_desc peut tre partitionn ! (il contient au moins 2 valeurs diffrentes)
        output: un tuple compos de 2 tuples
    """
    return ((m_desc[m_desc[:,n]<=s], m_class[m_desc[:,n]<=s]), \
            (m_desc[m_desc[:,n]>s], m_class[m_desc[:,n]>s]))

class ClassifierArbreNumerique(classif.Classifier):
    """ Classe pour reprsenter un classifieur par arbre de dcision numrique
    """
    
    def __init__(self, input_dimension, epsilon, LNoms=[]):
        """ Constructeur
            Argument:
                - intput_dimension (int) : dimension de la description des exemples
                - epsilon (float) : paramtre de l'algorithme (cf. explications prcdentes)
                - LNoms : Liste des noms de dimensions (si connues)
            Hypothse : input_dimension > 0
        """
        self.dimension = input_dimension
        self.epsilon = epsilon
        self.LNoms = LNoms
        # l'arbre est manipul par sa racine qui sera un Noeud
        self.racine = None
        
    def toString(self):
        """  -> str
            rend le nom du classifieur avec ses paramtres
        """
        return 'ClassifierArbreDecision ['+str(self.dimension) + '] eps='+str(self.epsilon)
        
    def train(self, desc_set, label_set):
        """ Permet d'entrainer le modele sur l'ensemble donn
            desc_set: ndarray avec des descriptions
            label_set: ndarray avec les labels correspondants
            Hypothse: desc_set et label_set ont le mme nombre de lignes
        """        
        self.racine = construit_AD_num(desc_set,label_set,self.epsilon,self.LNoms)
    
    def score(self,x):
        """ rend le score de prdiction sur x (valeur relle)
            x: une description
        """
        # cette mthode ne fait rien dans notre implmentation :
        pass
    
    def predict(self, x):
        """ x (array): une description d'exemple
            rend la prediction sur x             
        """
        return self.racine.classifie(x)

    def accuracy(self, desc_set, label_set):  # Version propre  aux arbres
        """ Permet de calculer la qualit du systme sur un dataset donn
            desc_set: ndarray avec des descriptions
            label_set: ndarray avec les labels correspondants
            Hypothse: desc_set et label_set ont le mme nombre de lignes
        """
        nb_ok=0
        for i in range(desc_set.shape[0]):
            if self.predict(desc_set[i,:]) == label_set[i]:
                nb_ok=nb_ok+1
        acc=nb_ok/(desc_set.shape[0] * 1.0)
        return acc

    def number_leaves(self):
        """ rend le nombre de feuilles de l'arbre
        """
        return self.racine.compte_feuilles()
    
    def affiche(self,GTree):
        """ affichage de l'arbre sous forme graphique
            Cette fonction modifie GTree par effet de bord
        """
        self.racine.to_graph(GTree)
